# My CRM

A full-featured CRM with Gmail integration, contacts, leads pipeline, and bulk import/export.

## Deploy to Vercel (via GitHub)

1. Create a free account at github.com
2. Create a new repository called "my-crm"
3. Upload ALL files from this folder (keep folder structure intact)
4. Go to vercel.com → Add New Project → Import your GitHub repo
5. Click Deploy (no config needed)
6. Add your Vercel URL to Google OAuth Authorized JavaScript Origins

## Run Locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173 and add it to your Google OAuth origins.
